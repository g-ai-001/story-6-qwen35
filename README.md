# 神话归来

**都市异能修仙小说 | 辰东《圣墟》式世界观**

## 简介

天地异变，灵气开始复苏。大学生楚风一夜之间从普通人觉醒为"进化者"，他发现远古的神话并非虚构，异界的大门正在打开。随着昆仑、蓬莱等秘境重现人间，万族降临，地球沦为战场。他必须在妖族、神族后裔的夹缝中求生，掠夺异宝，揭开人类并非地球唯一主角的残酷真相。

## 看点
- 辰东《圣墟》式的世界观
- 当代都市与上古神话的碰撞
- 氛围感极佳
- 热血升级与生死考验
- 跨越生死的爱情

## 故事概要

楚风是一名普通大学生，在灵气复苏的浪潮中觉醒为进化者。他结识了拥有凤瑶传承的苏瑶，两人携手闯荡昆仑、蓬莱等秘境，对抗深渊魔族、神族等强敌。在经历无数生死考验后，楚风发现自己竟是神族王子曦照的混血儿子，父亲为保护他而死。楚风觉醒终极太阳之力，击杀神族之主曦曜，为父报仇。最终，楚风与苏瑶战胜深渊魔族创始者魔天，封印深渊之门，在海边过上了幸福平静的生活。
