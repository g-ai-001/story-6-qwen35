# 神话归来

**都市异能修仙小说 | 辰东《圣墟》式世界观**

## 简介

天地异变，灵气开始复苏。大学生楚风一夜之间从普通人觉醒为"进化者"，他发现远古的神话并非虚构，异界的大门正在打开。随着昆仑、蓬莱等秘境重现人间，万族降临，地球沦为战场。他必须在妖族、神族后裔的夹缝中求生，掠夺异宝，揭开人类并非地球唯一主角的残酷真相。

## 看点
- 辰东《圣墟》式的世界观
- 当代都市与上古神话的碰撞
- 氛围感极佳
- 热血升级与生死考验

## 版本历史

### v0.6.0 (最新)
神族入侵篇完成，约5万字。神族大军入侵地球，楚风潜入敌营，揭露苏清雪的真实身份——神族公主曦和。楚风与曦和展开生死决战，领悟太阳凤凰炎。苏瑶为保护楚风牺牲，灵魂寄托于凤凰令中。楚风在绝境中觉醒太阳之光，击杀神族之主帝俊，神族战败。楚风踏上新的征程，准备追杀曦和并寻找复活苏瑶的方法。

秘境降临篇与元婴之路篇完成，共约9万字。楚风经历蓬莱仙境、龙宫秘境等冒险，正式加入守夜人组织，突破化神境。凤瑶残念彻底消散，留下最后的嘱托。

### v0.1.0 - v0.3.0
序章及前二十章发布，共约8万字。建立主角楚风人设，确立灵气复苏世界观，进入昆仑、蓬莱等秘境探索，为后续剧情埋下伏笔。
