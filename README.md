# 神话归来

**都市异能修仙小说 | 辰东《圣墟》式世界观**

## 简介

天地异变，灵气开始复苏。大学生楚风一夜之间从普通人觉醒为"进化者"，他发现远古的神话并非虚构，异界的大门正在打开。随着昆仑、蓬莱等秘境重现人间，万族降临，地球沦为战场。他必须在妖族、神族后裔的夹缝中求生，掠夺异宝，揭开人类并非地球唯一主角的残酷真相。

## 看点
- 辰东《圣墟》式的世界观
- 当代都市与上古神话的碰撞
- 氛围感极佳
- 热血升级与生死考验
