# 神话归来

**都市异能修仙小说 | 辰东《圣墟》式世界观**

## 简介

天地异变，灵气开始复苏。大学生楚风一夜之间从普通人觉醒为"进化者"，他发现远古的神话并非虚构，异界的大门正在打开。随着昆仑、蓬莱等秘境重现人间，万族降临，地球沦为战场。他必须在妖族、神族后裔的夹缝中求生，掠夺异宝，揭开人类并非地球唯一主角的残酷真相。

## 看点
- 辰东《圣墟》式的世界观
- 当代都市与上古神话的碰撞
- 氛围感极佳
- 热血升级与生死考验

## 版本历史

### v0.4.0 (最新)
秘境降临篇完成，约4万字。楚风从蓬莱回归，东海发生异变。龙宫秘境现世，神族与龙族爆发冲突。深渊魔族大祭司魔刹率军降临，东海陷入三族混战。楚风独闯龙宫，在凤瑶牺牲帮助下封印龙珠，阻止更大灾难发生。楚风正式加入守夜人组织，踏上新的征程。

### v0.3.0
蓬莱风云篇完成，约3.5万字。楚风进入蓬莱仙境，遭遇神族太子轩辕离。苏瑶现身，她是苏清雪的姐姐、凤凰旁支后裔。万族降临试炼之地，楚风遭神秘黑袍人追杀，凤瑶残魂显灵救主。楚风闯入凤凰神殿，揭开人类起源真相，获得《凰源真解》，修为虽暂时跌落但获得真正的凤凰意志传承。

### v0.2.0
昆仑秘境探索篇完成，约3万字。楚风觉醒凤凰血脉，获得凤瑶传承，击退化化神境妖族，获得凤凰令，开启新的征程。

### v0.1.0
序章及前五章发布，完成约1.5万字。建立主角楚风人设，确立灵气复苏世界观，为后续秘境探索埋下伏笔。
